# 🚦 Four-Way Traffic Signal Control System using Arduino UNO

## 📖 Project Description

This project presents a **Four-Way Traffic Signal Control System** developed using **Arduino UNO** and simulated in the **Wokwi Online Simulator**. The system demonstrates the operation of a road intersection by controlling traffic lights in four different directions using a predefined timing sequence. It serves as an educational project for understanding embedded systems, automation, and traffic management logic.

## 🔧 Project Concept

The intersection is divided into **four independent traffic directions**, each equipped with **Red, Yellow, and Green LEDs**. The Arduino UNO manages the signal transitions and ensures that only one direction receives the Green signal at any given time, while the remaining directions remain in the Red (STOP) state. Yellow signals are used during transitions to improve safety and realism.

## ⚙️ Working Principle

* Traffic movement is divided into **four directions**.
* Each direction is assigned a dedicated set of **Red, Yellow, and Green LEDs**.
* The Arduino executes a cyclic control sequence for all four directions.
* Only one direction is permitted to move at a time.
* The Yellow signal indicates a transition before switching to the next direction.
* The process repeats continuously to simulate a real-world traffic intersection.
